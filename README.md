\# Broken Cart Service



A deliberately buggy Spring Boot microservice for Agentic

DevOps training demonstrations.



\## The Bug

Discount codes are accepted but no discount is ever applied.



\## Demo Flow

1\. CI fails - two broken tests

2\. Create a GitHub Issue

3\. Assign to Copilot

4\. Review the agent's PR

5\. Merge and watch CI go green
